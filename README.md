# 🎓 Student Management System (Flask + SQLite)

## 📌 Description
A full-stack web application to manage student records using Flask and SQLite.  
It allows users to add, view, update, and delete student details through a simple and clean web interface.

---

## 🚀 Features
- ➕ Add Student  
- 📋 View Students  
- ✏️ Update Student  
- ❌ Delete Student  
- 🔔 Flash messages (success alerts)  
- 🎨 Clean UI using HTML & CSS  

---

## 🛠 Tech Stack
- Python  
- Flask  
- SQLite  
- HTML  
- CSS

## ▶️ How to Run

### 1. Install Flask
pip install flask

### 2. Create Database
python create_db.py
### 3. Run the Application
python app.py
### 4. Open in Browser
http://127.0.0.1:5000

### ⚠️ Requirements
Python installed on your system
Basic knowledge of terminal/command prompt
